在linux 7.1 下使用FTP 
1.配置yum 
2.安装FTP 服务软件。
```sh
yum install vsftpd
```
3.使用windows 的ftp 客户端访问linux FTP 

操作命令跟linux 命令一样
> get 命令是用于获取数据。
> put 是把文件进行上传。
